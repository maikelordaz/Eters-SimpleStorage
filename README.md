# ETHERS DEPLOYS

This is a solidity to demonstrate the usage of ethers and Ganache to deploy,
there are mor than one deploy contract to show different features

## :rocket: INSTALLATION

1. Clone this repo
2. Install the dependencies with
    > npm install
    > yarn install

## :floppy_disk: TECNOLOGIES

-   Ethers
-   Ganache
